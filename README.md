# Art Deco Pyramid Jewelry Box

An interactive showcase for the Twisted Genius Art Deco Pyramid Jewelry Box, designed to highlight its craftsmanship, multi-level storage concept, and statement-making artistic value. It provides customers with a rich, engaging experience while exploring the jewelry box online.

---

## Core Features

### 360Â° Product Viewer
Lets customers rotate and examine the jewelry box from every angle, revealing the metallic finishes and layered design.

### Layer-by-Layer Explorer
Visualizes the journey through four geometric tiers, each representing a compartment. Users can click each layer or navigate using buttons to uncover details about storage capacity, architectural motifs, and ideal item types for each section.

### Luxury Showcase Animation
Presents the jewelry box as a premium product, with fade-in and pulse effects, feature cards (e.g., "Art Deco Design," "Pyramidal Architecture"), specifications grid, and a bold call-to-action.
- Uses Twisted Genius brand colors (copper and teal) and elegant gradients to match the brand style.

### Customer Interaction
Customers can interactively explore storage options for necklaces, rings, and rare items by clicking through layers, hovering over features, and viewing high-resolution images.

- Feature highlights explain both artistic and functional aspects, supporting informed purchasing and brand storytelling.
